# injir_project

##  Краткое описание проекта

Веб-приложение для автоматизации процесса отслеживания и анализа хода приёмной кампании в вузе. Система позволяет загружать списки абитуриентов в разные даты, автоматически рассчитывать проходные баллы на основе поданных согласий, визуализировать динамику изменений и формировать итоговые отчёты.
Проект построен на **Flask** (Python) с использованием базы данных **SQLite** и ориентирован на локальное развертывание (например, в приёмной комиссии).



##  Инструкция по установке и развертыванию

### Предварительные требования
- **Python** версии 3.9 или выше.


### Шаг 1:
Распокавать архив в удобной для работы директории
### Шаг 2:
Создание окружения. Для создания окружения нужно прописать в терминал команды:
Windows:

python -m venv venv


venv\Scripts\activate

macOS / Linux:

python3 -m venv venv


source venv/bin/activate

### Шаг 3:
Установка зависимостей:

pip install -r requirements.txt

### Шаг 4: 
Запустите веб-сервер Flask командой:

python app.py
### Шаг 5: 
Доступ к приложению
Откройте браузер и перейдите по адресу:

http://127.0.0.1:5000
